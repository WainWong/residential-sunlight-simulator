# 住宅采光模拟器

一个面向普通用户的浏览器端住宅采光分析工具。无需安装 CAD，也无需上传户型数据：在页面中放置建筑、观察区和窗户，即可查看指定时刻的日照，并估算全天累计采光。

## 当前能力

- 三种住宅建筑模板与统一的观察功能
- 网格化观察区、普通窗、落地窗和阳台开口
- 按城市、日期、时间动态更新太阳位置和阴影
- 单时刻结果与全天累计日照估算
- 项目自动保存在本地，可导入、导出 JSON 和导出截图
- 桌面端、平板和手机端响应式界面
- 内置深圳冬至示例项目

> 当前结果适合早期方案比较和日常参考，不替代建筑规范审查或专业采光报告。

## 本地运行

需要 Node.js 22.12 或更高版本。

```bash
npm install
npm run dev
```

打开终端提示的本地地址即可使用。

## 测试与构建

```bash
npm test
npm run test:e2e
npm run build
```

## 部署到 Vercel

将仓库导入 Vercel 即可，项目已包含构建配置。也可以点击：

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone)

Vercel 会执行 `npm run build`，并发布 `dist` 目录。项目是纯静态前端，不需要数据库或服务器环境变量。

## 数据与隐私

项目数据默认只保存在当前浏览器的本地存储中。除非你主动导出文件，否则应用不会上传住宅信息。
