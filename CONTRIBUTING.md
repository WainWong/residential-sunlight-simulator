# 参与贡献

感谢你愿意改进住宅采光模拟器。

## 开始开发

1. Fork 仓库并创建一个短小、聚焦的分支。
2. 运行 `npm install` 安装依赖。
3. 使用 `npm run dev` 启动本地页面。
4. 为行为变化补充或更新测试。

提交前请运行：

```bash
npm test
npm run test:e2e
npm run build
```

## 提交建议

- 一个提交尽量只解决一个问题。
- 界面文字优先使用普通用户能理解的表达。
- 新交互需同时考虑鼠标、键盘和触屏。
- 改动模拟算法时，请说明假设、适用范围和验证方式。
- 不要提交真实住宅地址、私密户型或浏览器导出的个人项目。

## 报告问题

请尽量附上复现步骤、浏览器和设备信息、预期结果与实际结果；界面问题可以附截图。
